## Overview

Clone of the ALSA RAVENNA/AES67 driver originally provided by **Merging Technologies**. This Linux kernel module (LKM) enables the **RAVENNA** network audio transport protocol (compatible with **AES67**) within the ALSA subsystem on Linux.

RAVENNA/AES67 is a professional audio-over-IP transport standard used in studios, concert halls and broadcast installations.

## Features

- Linux kernel module (Loadable Kernel Module)
- RAVENNA/AES67 audio transport over IP
- Native ALSA integration
- Compatible with Merging Technologies equipment and other AES67 devices
