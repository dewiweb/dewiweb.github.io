## Overview

Full rewrite of the **Trajectoires** application, a web/mobile interface for controlling sound spatialisation via OSC. This modernised version is developed in TypeScript with a revised architecture.

Trajectoires lets you draw and animate trajectories for sound sources in space, then send them in real time to a spatialisation system.

## Features

- GUI for defining movement trajectories
- Real-time OSC output
- Modern TypeScript architecture
- Compatible with OSC systems (Holophonix, Max/MSP, etc.)
