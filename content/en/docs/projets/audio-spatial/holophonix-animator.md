## Overview

Desktop application for creating and managing **movement animations** for spatial audio tracks on the [Holophonix](https://holophonix.xyz/) processor.

It communicates with Holophonix via the **OSC** protocol and lets you define trajectories and dynamic behaviours for sound sources in 3D space.

## Features

- Create movement behaviours (circles, lines, free trajectories)
- Manage multiple tracks simultaneously
- OSC synchronisation with Holophonix
- Desktop GUI (Electron/Node.js)
