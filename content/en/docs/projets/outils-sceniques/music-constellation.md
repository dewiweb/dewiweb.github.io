## Overview

A web application that represents a **[beets](https://beets.io/)** music library as an **interactive constellation**. Each star is an album or artist, and the links between them illustrate relationships (genres, years, similarities).

A poetic, visual way to explore your music collection.

## Features

- Connect to a beets library (via the beets web API)
- SVG/Canvas constellation graph representation
- Interactive navigation (zoom, pan, select)
- Filter by genre, decade, artist

## Stack

- TypeScript
- Visualisation: D3.js or Canvas
- Backend: beets API
