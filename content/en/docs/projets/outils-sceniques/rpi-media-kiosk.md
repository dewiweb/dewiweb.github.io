## Overview

A plug-and-play solution to turn any **Raspberry Pi** into a **full-screen media kiosk** player. Automatically plays videos, PowerPoint presentations, PDFs and image slideshows from a USB stick or local storage â€” all rendered via **Chromium**.

Ideal for installations, exhibitions, digital signage or any stage use case requiring automatic media playback.

## Features

- Autoplay on boot
- Supports: video, PDF, PowerPoint (converted), image slideshows
- Load from USB stick or local storage
- Chromium in kiosk mode (full-screen, no UI)
- Simple configuration

## Requirements

- Raspberry Pi (any recent generation)
- Raspberry Pi OS
- Chromium (included by default on Raspberry Pi OS)
