## Overview

[Bitfocus Companion](https://bitfocus.io/companion) turns an Elgato Stream Deck into a professional control surface for broadcast and stage equipment. These modules integrate specific devices.

---

## companion-module-canon-network-legacy-http

**Repository** · [github.com/dewiweb/companion-module-canon-network-legacy-http](https://github.com/dewiweb/companion-module-canon-network-legacy-http)

Companion module for controlling **Canon cameras** via the legacy HTTP network API. Allows driving compatible Canon cameras from a Stream Deck.

- Language: JavaScript · Protocol: HTTP

---

## companion-module-powersoft-amplifier

**Repository** · [github.com/dewiweb/companion-module-powersoft-amplifier](https://github.com/dewiweb/companion-module-powersoft-amplifier)

Companion module for controlling **Powersoft amplifiers**. Monitor and control Powersoft amps (X series, Quattrocanali, etc.) from a Stream Deck.

- Language: TypeScript
