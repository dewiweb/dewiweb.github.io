## IanniX

**Fork** · [github.com/dewiweb/IanniX](https://github.com/dewiweb/IanniX)

**IanniX** is an open source graphical sequencer inspired by Iannis Xenakis, designed for digital art. It synchronises events and curves via OSC to real-time environments (Max/MSP, Pure Data, SuperCollider...).

- Language: C++ (Qt)
- Licence: GPL-3.0
- [Official site](http://www.iannix.org)

## scenic

**Fork** · [github.com/dewiweb/scenic](https://github.com/dewiweb/scenic)

**Telepresence** application for live performances and artistic installations. Allows sharing of video, audio and data streams between remote sites.

- Language: Python
- Licence: GPL-3.0
- [Documentation](http://code.sat.qc.ca/trac/scenic/)
