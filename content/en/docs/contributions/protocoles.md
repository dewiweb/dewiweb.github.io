## ember-plus

**Fork** · [github.com/dewiweb/ember-plus](https://github.com/dewiweb/ember-plus)

The official **Ember+** protocol from Lawo — an open control protocol for the broadcast and audiovisual industry. Used to communicate between broadcast equipment (mixers, routers, monitoring systems).

- Language: C++
- Licence: BSL-1.0
- [Original project](https://github.com/Lawo/ember-plus)

**Fork** · [github.com/dewiweb/ember-plus-sharp](https://github.com/dewiweb/ember-plus-sharp)

## OSCRouter

**Fork** · [github.com/dewiweb/OSCRouter](https://github.com/dewiweb/OSCRouter)

Simple UDP/TCP packet router for Mac and Windows with OSC-specific options. Redistributes OSC messages between multiple applications or devices.

- Language: C++
- Licence: MIT
