## trajectoires

**Fork** · [github.com/dewiweb/trajectoires](https://github.com/dewiweb/trajectoires)

Application mobile/web pour le contrôle de la spatialisation sonore via OSC. Fork de l'original, adapté et utilisé comme base de travail.

- Langage : Objective-C
- Usage : contrôle OSC de sources sonores depuis mobile

## SPARTA

**Fork** · [github.com/dewiweb/SPARTA](https://github.com/dewiweb/SPARTA)

Collection de plugins VST audio pour la production, reproduction et visualisation d'audio spatial. Suite développée par Aalto University.

- Langage : C++
- Licence : GPL-3.0
- [Projet original](http://research.spa.aalto.fi/projects/sparta_vsts/)
