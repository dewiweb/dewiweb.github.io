## ember-plus

**Fork** · [github.com/dewiweb/ember-plus](https://github.com/dewiweb/ember-plus)

Le protocole **Ember+** officiel de Lawo — protocole de contrôle ouvert pour l'industrie broadcast et audiovisuelle. Utilisé pour communiquer entre équipements de diffusion (mixeurs, routeurs, systèmes de monitoring).

- Langage : C++
- Licence : BSL-1.0
- [Projet original](https://github.com/Lawo/ember-plus)

## ember-plus-sharp

**Fork** · [github.com/dewiweb/ember-plus-sharp](https://github.com/dewiweb/ember-plus-sharp)

Implémentation Ember+ pour .NET, permettant de créer rapidement des consumers Ember+ entièrement typés dans n'importe quel langage .NET.

- Langage : C#
- Licence : BSL-1.0

## OSCRouter

**Fork** · [github.com/dewiweb/OSCRouter](https://github.com/dewiweb/OSCRouter)

Routeur de paquets UDP/TCP simple pour Mac et Windows, avec options spécifiques OSC. Permet de redistribuer des messages OSC entre plusieurs applications ou équipements.

- Langage : C++
- Licence : MIT
