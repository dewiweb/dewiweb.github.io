## IanniX

**Fork** · [github.com/dewiweb/IanniX](https://github.com/dewiweb/IanniX)

**IanniX** est un séquenceur graphique open source basé sur les travaux d'Iannis Xenakis, destiné à l'art numérique. Il synchronise via OSC des événements et des courbes vers des environnements temps réel (Max/MSP, Pure Data, SuperCollider...).

- Langage : C++ (Qt)
- Licence : GPL-3.0
- [Site officiel](http://www.iannix.org)

## scenic

**Fork** · [github.com/dewiweb/scenic](https://github.com/dewiweb/scenic)

Application de **téléprésence** pour les performances live et les installations artistiques. Permet de partager flux vidéo, audio et données entre sites distants.

- Langage : Python
- Licence : GPL-3.0
- [Documentation](http://code.sat.qc.ca/trac/scenic/)
