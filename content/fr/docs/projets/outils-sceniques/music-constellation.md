## Présentation

Application web qui représente une bibliothèque musicale gérée par **[beets](https://beets.io/)** sous forme de **constellation interactive**. Chaque étoile est un album ou un artiste, et les liens entre elles illustrent les relations (genres, années, similarités).

Une façon poétique et visuelle d'explorer sa collection musicale.

## Fonctionnalités

- Connexion à la bibliothèque beets (via l'API web de beets)
- Représentation en graphe de constellation SVG/Canvas
- Navigation interactive (zoom, déplacement, sélection)
- Filtrage par genre, décennie, artiste

## Stack technique

- TypeScript
- Visualisation : D3.js ou Canvas
- Backend : API beets
