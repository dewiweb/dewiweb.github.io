## Présentation

[Bitfocus Companion](https://bitfocus.io/companion) est un logiciel qui transforme une Elgato Stream Deck en surface de contrôle professionnelle pour des équipements broadcast et scéniques. Ces modules permettent d'intégrer des équipements spécifiques.

---

## companion-module-canon-network-legacy-http

**Dépôt** · [github.com/dewiweb/companion-module-canon-network-legacy-http](https://github.com/dewiweb/companion-module-canon-network-legacy-http)

Module Companion pour le contrôle de **caméras Canon** via l'API HTTP réseau (version legacy). Permet de piloter les caméras Canon compatibles depuis une Stream Deck.

- Langage : JavaScript
- Protocole : HTTP

---

## companion-module-powersoft-amplifier

**Dépôt** · [github.com/dewiweb/companion-module-powersoft-amplifier](https://github.com/dewiweb/companion-module-powersoft-amplifier)

Module Companion pour le contrôle d'**amplificateurs Powersoft**. Permet de monitorer et contrôler les amplis Powersoft (série X, Quattrocanali, etc.) depuis une Stream Deck.

- Langage : TypeScript
