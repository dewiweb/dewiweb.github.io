## Présentation

Application desktop permettant de créer et gérer des **animations de mouvement** pour les pistes audio spatiales du processeur [Holophonix](https://holophonix.xyz/).

Elle communique avec Holophonix via le protocole **OSC** et permet de définir des trajectoires et comportements dynamiques pour les sources sonores dans l'espace 3D.

## Fonctionnalités

- Création de comportements de mouvement (cercles, lignes, trajectoires libres)
- Gestion de plusieurs pistes simultanées
- Synchronisation OSC avec Holophonix
- Interface graphique desktop (Electron/Node.js)
