## Présentation

Refonte complète de l'application **Trajectoires**, une interface web/mobile pour le contrôle de la spatialisation sonore via OSC. Cette version modernisée est développée en TypeScript avec une architecture revue.

Trajectoires permet de dessiner et animer des trajectoires de sources sonores dans l'espace, puis de les envoyer en temps réel vers un système de spatialisation.

## Fonctionnalités

- Interface graphique pour définir des trajectoires de mouvement
- Envoi en temps réel via OSC
- Architecture moderne TypeScript
- Compatible avec les systèmes OSC (Holophonix, Max/MSP, etc.)
