## Présentation

Clone du driver ALSA RAVENNA/AES67 originellement fourni par **Merging Technologies**. Ce driver noyau Linux (LKM) permet d'utiliser le protocole de transport audio réseau **RAVENNA** (compatible **AES67**) avec l'architecture ALSA sous Linux.

RAVENNA/AES67 est un standard de transport audio professionnel sur réseau IP, utilisé dans les studios, salles de concert et installations de diffusion.

## Fonctionnalités

- Driver noyau Linux (Loadable Kernel Module)
- Transport audio RAVENNA/AES67 sur réseau IP
- Intégration native avec ALSA
- Compatible avec les équipements Merging Technologies et autres appareils AES67
